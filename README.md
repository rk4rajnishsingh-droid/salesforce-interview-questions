## 🎯 Sample Questions

### Apex Triggers
- How do you prevent duplicate records using an Apex trigger?
- What is the difference between before and after triggers?
- How do you prevent recursion in Apex triggers?
- What is bulkification and why does it matter?

### SOQL
- What is the difference between WHERE and HAVING?
- How do you write parent-to-child and child-to-parent queries?
- What are SOQL Date Literals and when do you use them?
- Why doesn't SOQL support SELECT *?

### Salesforce Flows
- What is the difference between Before Save and After Save flows?
- How do you prevent recursion in flows?
- What is $Record vs $Record__Prior?
- How do you handle faults in flows?

### Agentforce
- What is an AI Agent in Salesforce?
- What is the Einstein Trust Layer?
- How do Topics and Actions work together?
- What is Retrieval Augmented Generation (RAG)?
